cloud computing :-
cloud computing is the delivery of computing services such as servers ,database ,storage 
networking  , and software over the internet. 
instead of buying and maintaining physical hardware , users can access resources on demand 
and pay only for what they  use.


IAAS:- infrastructure as a service.
iaas provides virtualized computing resources over the internet .
the cloud provides :- physical servers 
                      storage 
                      networking 
                      data centers 
the user manages :- operating system
                    application
                    data
e.g :- azure virtual machines 
       amazon Ec2

PAAS :- platform as a service
paas provides platform for developing and deploying and testing applications .
the  cloud provides :- hardware 
                        operating systems 
                        runtime environment 
the user manages :-application  code 
                    data 
e.g. :- azure app service 
        google app engine 

SAAS:- software as a service 
saas provides ready to use software over the internet .
the cloud manages everything .
the user only uses the application 
e.g. :- gmail
        google docs 
        microsoft 365
        zoom
