01 PUBLIC CLOUD  ;- 

public cloud is owned and operated by a third partyb cloud provider . resources are shared to public

advantages of public cloud is :- low cost
                                easy scalibilty

        amazon web service 
        google cloud platform 
  
  02 PRIVATE CLOUD :- 

  private cloud is dedicated to a single organization  . resources are not shared with other customers .
  advantages of privat cloud ;- more control
                                better security 
                                customization 
disadvantage of private cloud :- higher cost 
                                moree management is required 
e.g. :- an organization's own data center .

03 HYBRID CLOUD :- 
hybrid cloud means mix of both cloud . 
it keeps sensitive data into private cloud and use public cloud resouces when it needed . 
advantages of hybrid cloud :- flexibility 
                            better cost management 
                            improved security 
 e.g. :- store customer data in a private cloud and host a website in azure .


 in short :- 
 public cloud :- shared resources 
 private cloud :- dedicated resources 
 hybrid cloud :- combination of both
